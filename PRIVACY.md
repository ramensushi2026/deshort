# Privacy Policy

**deshort** does not collect, transmit, or sell any personal data.

## Data collection
- No analytics
- No tracking
- No account / login
- No cookies set by the extension

## What the extension does
- Runs locally in your browser as a content script.
- Hides specific UI modules (e.g. YouTube Shorts / Facebook Reels shelves) by applying CSS and DOM updates.

## Data access
The extension needs access to the pages it runs on in order to hide those UI elements. All processing happens **on-device**.
