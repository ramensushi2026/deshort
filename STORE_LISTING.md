# Chrome Web Store listing draft

## Title
**deshort**

## Short description (132 chars max)
Hide YouTube Shorts and Facebook Reels from feeds and listings (direct links still work).

## Detailed description
Tired of short-form video taking over your feed?

deshort removes *Shorts/Reels listing modules* so you can browse YouTube and Facebook with fewer distractions.

What it does:
- Hides YouTube Shorts shelves and Shorts entry points in the sidebar
- Hides Shorts cards in YouTube search results
- Hides Facebook Reels / short-video modules in feeds

What it does NOT do:
- It does not block direct Shorts/Reels URLs
- It does not collect or transmit any data

Privacy policy:

## Permission justification (for review)
The extension runs content scripts on:
- youtube.com / m.youtube.com
- facebook.com / m.facebook.com

This access is used only to hide specific UI elements (DOM/CSS) locally.
