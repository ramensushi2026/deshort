deshort.com（本機自用 extension）

用途：
- YouTube：隱藏 Shorts 架 / Shorts 列表區塊 / 部分左邊 Shorts 入口
- Facebook：隱藏 Reels/短片 相關模組（Feed 內、部分左邊入口）
- 只係「睇唔到 listing」，唔會 block 你直接入 shorts/reels 連結。

Chrome 安裝（Unpacked / 開發者模式）：
1) Chrome 打開：chrome://extensions/
2) 右上角開「Developer mode / 開發者模式」
3) 按「Load unpacked / 載入未封裝項目」
4) 揀呢個資料夾：
   /Users/ramen/code/ramen-hide-shorts-reels-extension
5) 之後開 YouTube / Facebook refresh 一下。

Safari：
- Safari 要用「Safari Web Extension」方式（要 Xcode）。
- 如果你想我幫你轉成 Safari 可用版本：同我講聲「要 Safari 版」，我會用你部 Mac 直接轉。

備註：
- YouTube/Facebook 會不時改 UI，萬一某日又見到 Shorts/Reels，話我知，我再更新 selectors。
